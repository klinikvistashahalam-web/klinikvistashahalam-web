modalImg.src = image URL\n  - modalTitle.textContent = title\n  - modalDesc.textContent = description\n  - modalBenefits.innerHTML = benefits list\n        ↓\nShow modal with animation\n        ↓\nCustomer reads information\n        ↓\nCustomer closes:\n  ├─ Click X button\n  ├─ Press Escape\n  └─ Click outside\n        ↓\ncloseServiceModal()\n        ↓\nHide modal\n```\n\n---\n\n## Data Example: Pap Smear\n\n```javascript\n'service-pap-smear.jpg': {\n    title: 'Pap Smear',\n    \n    description: 'Pap Smear adalah tes skrining untuk mendeteksi sel abnormal '\n                 'dan kanker serviks pada wanita. ↓\nopenServiceModal(element) ──┐\n                              ├─→ Get image filename\n                              ├─→ Find in serviceData\n                              ├─→ Set modal.display = 'block'\n                              └─→ Prevent body scroll\n        ↓\nModal Appears with Animation\n        ↓\nClick X / Press Escape / Click Outside\n        ↓\ncloseServiceModal() ─────────┐\n                             ├─→ Set modal.display = 'none'\n                             └─→ Enable body scroll\n        ↓\nModal Disappears\n```\n\n---\n\n## CSS Animations\n\n### Modal Open
